Arhopala ID

Interactive identification key for Peninsular Malaysian Oak Blue butterflies

What is this app?

This tool walks you through a step-by-step morphological key to identify species in the genus Arhopala — the Oak Blues — recorded from Peninsular Malaysia. Answer the questions about the butterfly in front of you; the key will narrow down to a species or group.

How to use it

  1. Start from the first question card and select the best matching character.
  2. Use the Back button to correct a previous choice.
  3. If you cannot observe the required character, select Cannot determine — the key will route you to the best available result.
  4. A result card may identify a single species or flag an unresolved pair requiring additional observation (e.g. upperside colour or genitalia).
  5. Tap Start Over at any time to reset.

Feature scoring mode

If multiple features are visible in your photo but you want to skip the step-by-step flow, try Feature Scoring. Mark only the characters you can observe; all species are scored and ranked in real time. Unanswered features never penalise a candidate.

Coverage

Scope: Peninsular Malaysia only. The key is based on identification literature covering Arhopala species recorded from the Malay Peninsula. Bornean endemic species are not covered. Characters used are primarily from the underside wing pattern and morphology, supplemented by geography where Peninsula and Bornean species overlap.

⚠ Some species pairs cannot be reliably distinguished without observing the male upperside or performing genitalia dissection; the key will flag these cases explicitly.

Species search

Switch to the Species tab to browse or search all taxa covered by this key. Each species page shows identification characters, photos (from iNaturalist), and the complete decision path that leads to it.

Data & sources

The identification key is based on published morphological literature for Malaysian Arhopala. Species occurrence data and photographs are sourced from iNaturalist (CC-licensed).
